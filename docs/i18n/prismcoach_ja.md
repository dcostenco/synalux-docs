# 🏋️ PrismCoach

**ミリタリーグレードのAIフィットネスコーチ。** iPhone · iPad · Apple Watch アプリ。回復状態を追跡し、疲労を予測し、トレーニングプログラムを生成し、23言語でリアルタイムにコーチング。デバイス上のAIモデル（Pro+）により完全オフラインで動作。独立リポジトリ: [`forge-watch`](https://github.com/dcostenco/forge-watch)。

## 🔋 ボディバッテリー

あなたの主要なレディネススコア — 夜間HRV、安静時心拍数のトレンド、睡眠の質、過去7日間に蓄積されたトレーニング負荷の複合指標。

*   **スコアリング** — 0〜100。≥ 75 = フレッシュ（緑）、50〜74 = 普通（黄）、25〜49 = 疲労（オレンジ）、< 25 = 消耗（赤）。
*   **HealthKit連携** — Apple Watchが取得した夜間のパッシブHRVを読み取る。手動入力は不要。
*   **ベースライン再校正** — ボディバッテリーは週次でローリングベースラインを再計算するため、熟練したアスリートと初心者のどちらも正しく正規化されたスコアが表示される。
*   **ATRエンジン** — Adaptive Training Readinessが7種類以上のバイオメトリクス信号を単一のレディネス指数に統合。急性負荷（過去3日間）と慢性負荷（28日間）の比率を考慮。

## 💪 筋肉回復マップ

SwiftUIでレンダリングされたパラメトリックボディマップキャンバスを使用して、14の解剖学的部位にわたる筋肉ごとの疲労追跡。

*   **14の筋肉グループ** — 胸、前部/側部/後部デルト、上腕二頭筋、上腕三頭筋、僧帽筋、広背筋、体幹、腰部、臀部、大腿四頭筋、ハムストリングス、ふくらはぎ。
*   **チャージモデル** — 各筋肉は運動量とRPEに応じて、トレーニング後48〜96時間かけて100%から0%へと減衰。回復はシグモイド曲線（非線形）に従う。
*   **前面/背面切り替え** — タップでシルエットを反転。iOSはセグメントピッカーを使用、watchOSはスワイプタブとして両面を表示。
*   **カラーコーディング** — ≥ 75% 緑、50〜75% 黄、25〜50% オレンジ、< 25% 赤。
*   **トレーニングハイライト** — 本日のプログラムのターゲット筋肉がキャンバス上でオレンジにパルス表示。

## 📋 トレーニングプログラム

主要なトレーニング目標をすべてカバーする、科学に基づく6つのピリオダイゼーションテンプレート。

| プログラム | 構成 | 目標 |
|---|---|---|
| PPL (Power/Push/Pull/Legs) | 6日間上半身/下半身スプリット | 筋肥大 + 筋力 |
| 5/3/1 Wendler | 4日間バーベル + アクセサリー | パワーリフティング筋力 |
| GZCLP | 3日間ティアシステム | 初心者線形進行 |
| 上半身/下半身 | 4日間クラシックスプリット | バランスの取れた筋肥大 |
| 全身 | 3日間コンパウンド中心 | 一般的なフィットネス |
| デロード / メンテナンス | 1日アクティブリカバリー | 再生 |

*   **AI生成プログラム**（Elite）— 目標と制約を説明するだけで、PrismCoachがPrism 8BまたはClaude Sonnetを使用して完全カスタムの複数週間ブロックを生成。
*   **JSONプログラムジェネレーター** — プログラムはTyped Swiftモデルとして表現され、エンジンはデバイス上で < 500 ms で8週間の完全プログラムを生成可能。
*   **Watchとの同期** — アクティブプログラムはオフラインコーチング用にApple Watchにキャッシュされる（ペイロード < 100 KB）。

## 🍎 ニュートリションエンジン

NLP駆動の食事記録 — 普通の言葉で食べ物を説明するだけでマクロ栄養素を取得。

*   **NLP食事記録** — 「卵2個、バター付きトースト、ブラックコーヒー」と入力または口述するだけで、エンジンが正規表現 + Claude Haikuを使って食品エンティティ、量、単位を解析。
*   **マクロ目標** — 体重、目標（減量/増量/維持）、活動レベルから計算。トレーニング負荷に基づいて毎日調整。
*   **カロリー追跡** — 内訳付きの日次累計（タンパク質 / 炭水化物 / 脂質 / 食物繊維 / 水分）。
*   **栄養密度スコアリング** — 記録された食品に基づいてマイクロ栄養素の不足を検出。
*   **水分補給リマインダー** — トレーニング中の発汗量推定値に基づくアダプティブプッシュ通知。

## 🤖 AIコーチ

回復状態、前回のセッション、プログラムコンテキストを把握した会話型コーチング。

*   **無料プラン** — AI機能なし（静的プログラムテンプレートのみ）。
*   **Pro — デバイス上のPrism 1.7B** — Metal加速付きのllama.cppで動作。ネットワーク不要、完全プライベート。空きRAMが1,600MB以上の場合はPrism Coder 1.7B（オンデバイス、Metal）を使用。空きRAMが450〜799MBの場合はSmolLM2-360M（185MB、オンデバイス）を使用。コーチングは常に機能し、劣化するだけ。トレーニングに関する質問に回答し、フォーム修正を提案し、レディネスに基づいて本日のボリュームを調整。
*   **Elite — Prism 1.7B → 8B → Claude Sonnet カスケード** — まずデバイス上で実行、複雑な質問にはPrism推論サーバー（8B）にエスカレート、より細やかなコーチングにはClaude Sonnetにフォールバック。
*   **コンテキストウィンドウ** — ForgeMemoryStoreが直近3セッション、アクティブなプログラム週、現在の筋肉チャージ、ボディバッテリースコアを全プロンプトに注入。
*   **音声出力** — AIの返答はForgeTTSEngineで読み上げられる: Synalux クラウドTTS（MP3、24 kHz）とAVSpeechSynthesizerオフラインフォールバック。6つのコーチングトーン: フレンドリー、穏やか、エキサイティング、精確、共感的、希望に満ちた。
*   **プロアクティブコーチング** — ProactiveCoachEngineが7種類のトリガーに基づいて自動的にインサイトを提示（例：「HRVが15%低下しました — 今日は強度を下げることを検討してください」）。

## ⌚ Apple Watchアプリ

完全なコンパニオンアプリ — 通知だけではない。手首での独立したセッション追跡。

*   **6つのWatchタブ** — ダッシュボード（ボディバッテリー）、筋肉マップ、ワークアウトロガー、Watch AIコーチ、CNSタップテスト、設定。
*   **Watch AIコーチ** — 手首で使える完全な会話型AI：クエリはWatchConnectivity経由でペアリングしたiPhoneに中継され、iPhoneがオンデバイス1.7B → 360M → クラウドのカスケードを実行して応答を返信。iPhoneに接続できない場合は明確な「iPhoneに到達できません」状態にフォールバック。
*   **ワークアウトセッション** — 手首から直接セット（種目、重量、回数、RPE）を記録。ハプティックカウントダウン付き90秒レストタイマー。
*   **CNSタップテスト** — トレーニング前の10秒間の高速タップテスト。タップ/秒を測定し、個人ベースラインを下回った場合に神経筋疲労を検出。
*   **ハプティックペースキーパー** — AMRAP/EMOMサーキット中のリズムハプティクス。
*   **スマートフォンとの同期** — WatchConnectivityブリッジがバッテリー、ボディバッテリー、機能フラグをリアルタイムで双方向に転送。
*   **自動セット検出** — 加速度センサー + ジャイロスコープがセット開始/終了を認識し、種目タイプを分類。Watchバッテリー節約のために無効化可能。
*   **速度ベーストレーニング** — CoreMotionを使用した手首装着型バー速度推定。バー速度が15%超低下した場合に神経筋疲労を検出。

## 🧬 フェムエンジン

女性アスリート向けの月経周期フェーズ対応メトリクス調整（オプトイン）。

*   **4つのフェーズ** — 月経期、卵胞期、排卵期、黄体期。
*   **トレーニング調整** — ボリューム、強度目標、RPE推奨値が、筋力と回復に対するホルモン効果に関する発表済み研究に基づいてフェーズごとに変化。
*   **ボディバッテリー補正** — 基礎体温とHRVのベースラインがフェーズごとに調整され、サイクル中盤の誤った疲労フラグを防止。
*   **プライバシー** — 周期データはデバイスから出ることはない（CoreData、同期なし）。

## 🧠 睡眠とHRV

バックグラウンドで静かに動作する夜間回復サイエンス。

*   **HRVキャプチャ** — 夜間HRVサンプルのためのパッシブHealthKitクエリ。アクティブな測定は不要。
*   **睡眠ステージ分析** — Appleの睡眠ステージデータ（覚醒/コア/深睡眠/REM）を読み取って睡眠の質を評価。
*   **トレーニングループ** — SleepTrainingLoopが前日の負荷と翌朝のHRVを相関させ、時間をかけて個人的なストレス-回復モデルを構築。
*   **ボディバッテリーベースライン** — 個人基準として使用する28日間のHRV中央値、偏差がバッテリースコアを駆動。

## 🔊 音声とTTS

あなたの言語でAIコーチングを音声で。

*   **Synalux TTS** — Synaluxポータル経由の24 kHz/96 kbpsモノラルクラウドMP3合成。6つのコーチングトーン。
*   **オフラインフォールバック** — 利用可能な最高音声品質を自動選択するAVSpeechSynthesizer（プレミアム ≥ 拡張 ≥ デフォルト）。23のサポート言語すべてで動作。
*   **音楽ダッキング** — ForgeTTSEngineがコーチングスピーチ中にApple Music / Spotifyを自動的にダッキングし、終了後に音量を復元。
*   **自動トーン推論** — メッセージ内容が感情的キーワード（PR、疲労、怪我、カムバックなど）でスキャンされ、適切なトーンが自動的に選択。
*   **音声コマンド** — VoiceCommandEngineがSpeechフレームワークを通じてハンズフリーのセット記録（「100kg 5回 RPE 8 記録」）とナビゲーション（「筋肉マップを表示」）を処理。

## 🏗️ アーキテクチャ

*   **PrismCoachCore** — すべてのエンジン、モデル、ビジネスロジックを含む共有Swiftパッケージ（SPM）。iOSアプリ、Mac Catalystアプリ、watchOS拡張機能で使用。
*   **llama.cpp (Metal)** — ローカルSPMパッケージ（`_llama_cpp_local`）を通じたPrism 1.7B GGUFモデルのデバイス上推論。iOS/macOSのみ、コンパイル時にwatchOSから除外。
*   **WatchConnectivityブリッジ** — `WatchBridge`が筋肉バッテリー、ボディバッテリー、機能フラグをiPhoneとWatch間でリアルタイムに双方向同期。
*   **CloudKit同期** — `CloudKitSyncEngine`がCKRecordチェンジトークンと競合解決を使用してデバイス間でワークアウト履歴とユーザープロフィールを複製。
*   **HealthKit** — HRV、睡眠ステージ、安静時HR、アクティブエネルギー、ワークアウトサンプルを読み取り。ワークアウトセッションサマリーを書き込み。
*   **Siri Intents** — ショートカット統合のための`LogWorkoutIntent`、`GetBodyBatteryIntent`、`StartRestTimerIntent`。

## 🔒 プライバシー

*   **ローカルファースト** — すべてのバイオメトリクスデータはデバイス上に保持。分析SDKなし。サードパーティクラッシュレポートなし。
*   **HealthKit** — ワークアウトセッション書き込みを除き読み取り専用アクセス。App Storeプライバシーラベルに記載。
*   **AIプロンプト** — Proプランはいかなるサーバーにもデータを送信しない。Eliteプランは匿名化されたトレーニングコンテキストをPrism推論サーバーに送信（PII不含、HealthKitデータ不含）。
*   **サブスクリプション確認** — `api.prismcoach.app/subscription/check`へHTTPS経由でメールを送信。メールはKeychainに24時間TTLと48時間猶予期間で保存。
*   **周期データ** — フェムエンジンのデータはCoreDataのみ、同期なし。
