## Fonctionnalités

- **Caisse — disposition 3 panneaux, scan code-barres, gestion des places**
- **Affichage Cuisine (KDS) — tickets, comptage journalier, code couleur par type**
- **Paiements — 6 méthodes : carte, espèces, carte cadeau, compte maison, partage, addition bar**
- **Rapports — 8 onglets : ventes, PMIX, ingénierie menu, vitesse, serveur, paiements, annulations, travail**
- **Personnel — PIN, 7 niveaux RBAC, pointage, pauses, planning, pourboires, paie**
