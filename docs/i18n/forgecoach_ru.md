# 🏋️ ForgeCoach

**ИИ-тренер по фитнесу военного класса.** Приложение для iPhone · iPad · Apple Watch. Отслеживает восстановление, прогнозирует усталость, генерирует программы тренировок и тренирует в реальном времени — на 23 языках. Полностью работает офлайн благодаря модели ИИ на устройстве (Pro+). Отдельный репозиторий: [`forge-watch`](https://github.com/dcostenco/forge-watch).

---

## 🔋 Заряд тела

Ваш основной показатель готовности — комбинация ночного HRV, тенденции ЧСС в покое, качества сна и нагрузки, накопленной за последние 7 дней.

*   **Оценка** — 0–100. ≥ 75 = Свежий (зелёный), 50–74 = Умеренный (жёлтый), 25–49 = Уставший (оранжевый), < 25 = Истощённый (красный).
*   **Интеграция с HealthKit** — считывает пассивный ночной HRV, зафиксированный Apple Watch. Ручной ввод не требуется.
*   **Перекалибровка базовой линии** — заряд тела еженедельно пересчитывает скользящую базовую линию, поэтому хорошо тренированный спортсмен и новичок видят правильно нормализованные оценки.
*   **Двигатель ATR** — Adaptive Training Readiness синтезирует 7+ биометрических сигналов в единый индекс готовности. Учитывает соотношение острой (последние 3 дня) и хронической (28 дней) нагрузки.

## 💪 Карта восстановления мышц

Отслеживание усталости каждой мышцы в 14 анатомических зонах с помощью параметрического холста карты тела, отрисованного в SwiftUI.

*   **14 мышечных групп** — Грудь, Передний/Боковой/Задний дельты, Бицепсы, Трицепсы, Трапеции, Широчайшие, Кор, Поясница, Ягодицы, Квадрицепсы, Подколенные сухожилия, Икры.
*   **Модель заряда** — каждая мышца убывает со 100% до 0% в течение 48–96 часов после тренировки в зависимости от объёма нагрузки и RPE. Восстановление следует сигмоидной кривой (нелинейной).
*   **Переключение перед/зад** — нажмите, чтобы перевернуть силуэт. iOS использует сегментированный переключатель; watchOS показывает оба вида как вкладки со свайпом.
*   **Цветовое кодирование** — ≥ 75% зелёный, 50–75% жёлтый, 25–50% оранжевый, < 25% красный.
*   **Выделение тренировки** — мышцы, на которые ориентирована программа сегодня, пульсируют оранжевым на холсте.

## 📋 Программы тренировок

Шесть научно обоснованных шаблонов периодизации, охватывающих все основные цели тренировок.

| Программа | Структура | Цель |
|---|---|---|
| PPL (Сила/Жим/Тяга/Ноги) | 6-дневный сплит верх/низ | Гипертрофия + сила |
| 5/3/1 Wendler | 4 дня штанга + вспомогательные | Сила в пауэрлифтинге |
| GZCLP | 3-дневная система уровней | Линейный прогресс для новичков |
| Верх/Низ | 4-дневный классический сплит | Сбалансированная гипертрофия |
| Всё тело | 3 дня с акцентом на базовые | Общая физическая подготовка |
| Разгрузка / Поддержание | 1 день активного восстановления | Регенерация |

*   **Программы, сгенерированные ИИ** (Elite) — опишите свою цель и ограничения; ForgeCoach генерирует полностью кастомный многонедельный блок с помощью Prism 8B или Claude Sonnet.
*   **Генератор программ JSON** — программы представлены в виде типизированных Swift-моделей; движок может сгенерировать полную 8-недельную программу менее чем за 500 мс на устройстве.
*   **Синхронизация с Watch** — активная программа кэшируется на Apple Watch для офлайн-тренировок (размер пакета < 100 КБ).

## 🍎 Двигатель питания

Дневник питания на основе NLP — описывайте еду на естественном языке и получайте макронутриенты.

*   **Запись приёма пищи через NLP** — введите или продиктуйте «2 яйца, тост с маслом, чёрный кофе» и движок анализирует пищевые сущности, количества и единицы с помощью регулярных выражений + Claude Haiku.
*   **Цели по макронутриентам** — рассчитываются на основе веса тела, цели (дефицит/профицит/поддержание) и уровня активности. Корректируются ежедневно с учётом тренировочной нагрузки.
*   **Подсчёт калорий** — нарастающий дневной итог с разбивкой (белки / углеводы / жиры / клетчатка / вода).
*   **Оценка плотности питательных веществ** — указывает на дефицит микронутриентов на основе записанных продуктов.
*   **Напоминания о гидратации** — адаптивные push-уведомления, основанные на оценке темпа потоотделения во время тренировки.

## 🤖 ИИ-тренер

Разговорный коучинг, который знает ваше состояние восстановления, последнее занятие и контекст программы.

*   **Бесплатный уровень** — без ИИ (только статичные шаблоны программ).
*   **Pro — Prism 1.7B на устройстве** — работает через llama.cpp с Metal-ускорением. Без сети, полностью конфиденциально. Отвечает на вопросы о тренировках, предлагает исправления техники, корректирует объём сегодняшнего занятия на основе готовности.
*   **Elite — каскад Prism 1.7B → 8B → Claude Sonnet** — сначала на устройстве; для сложных вопросов эскалирует к серверу вывода Prism (8B); при необходимости тонкого коучинга переключается на Claude Sonnet.
*   **Контекстное окно** — ForgeMemoryStore вводит последние 3 занятия, активную неделю программы, текущие заряды мышц и оценку заряда тела в каждый промпт.
*   **Голосовой вывод** — ответы ИИ озвучиваются через ForgeTTSEngine: Synalux cloud TTS (MP3, 24 кГц) с AVSpeechSynthesizer как офлайн-резервом. Шесть тонов тренировки: Дружелюбный, Спокойный, Воодушевлённый, Точный, Эмпатичный, Полный надежды.
*   **Проактивный коучинг** — ProactiveCoachEngine выдаёт незапрошенные подсказки (например, «Ваш HRV снизился на 15% — рассмотрите снижение интенсивности сегодня») на основе 7 типов триггеров.

## ⌚ Приложение Apple Watch

Полноценное приложение-компаньон — не просто уведомления. Независимое отслеживание сессий на запястье.

*   **5 вкладок Watch** — Панель управления (Заряд тела), Карта мышц, Журнал тренировок, Тест нажатий ЦНС, Настройки.
*   **Тренировочная сессия** — записывайте подходы (упражнение, вес, повторения, RPE) прямо с запястья. 90-секундный таймер отдыха с гаптическим обратным отсчётом.
*   **Тест нажатий ЦНС** — 10-секундный тест быстрых нажатий перед тренировкой. Измеряет нажатий/сек; сигнализирует о нейромышечной усталости, если показатель ниже личной базовой линии.
*   **Гаптический метроном** — ритмичная гаптическая вибрация во время схем AMRAP/EMOM.
*   **Синхронизация с телефоном** — мост WatchConnectivity двунаправленно передаёт заряды, заряд тела и флаги функций в реальном времени.
*   **Автоопределение подходов** — акселерометр + гироскоп распознают начало/конец подхода и классифицируют тип упражнения. Отключите для экономии заряда Watch.
*   **Тренировка на основе скорости** — оценка скорости штанги на запястье с помощью CoreMotion. Сигнализирует о нейромышечной усталости при падении скорости штанги > 15%.

## 🧬 Двигатель Femme

Коррекция метрик с учётом фазы цикла для спортсменок (по желанию).

*   **4 фазы** — Менструальная, Фолликулярная, Овуляторная, Лютеиновая.
*   **Коррекция тренировок** — объём, целевые значения интенсивности и рекомендации по RPE меняются в зависимости от фазы на основе опубликованных исследований о гормональных эффектах на силу и восстановление.
*   **Коррекция заряда тела** — базальные значения температуры тела и ВСР корректируются по фазе, чтобы предотвратить ложные сигналы усталости в середине цикла.
*   **Конфиденциальность** — данные цикла никогда не покидают устройство (CoreData, без синхронизации).

## 🧠 Сон и HRV

Наука ночного восстановления, работающая в тишине в фоновом режиме.

*   **Захват HRV** — пассивные запросы HealthKit для ночных образцов HRV. Активное измерение не требуется.
*   **Анализ фаз сна** — считывает данные фаз сна Apple (бодрствование/лёгкий/глубокий/REM) для оценки качества сна.
*   **Тренировочный цикл** — SleepTrainingLoop сопоставляет нагрузку предыдущего дня с HRV следующего утра для построения персональной модели стресс-восстановление со временем.
*   **Базовая линия заряда тела** — скользящая медиана HRV за 28 дней как личный ориентир; отклонения определяют оценку заряда.

## 🔊 Голос и TTS

ИИ-коучинг, произносимый на вашем языке.

*   **Synalux TTS** — облачный синтез MP3 при 24 кГц/96 кбит/с моно через портал Synalux. 6 тонов тренировки.
*   **Офлайн-резерв** — AVSpeechSynthesizer с автоматическим выбором наивысшего доступного качества голоса (премиум ≥ расширенный ≥ стандартный). Работает на всех 23 поддерживаемых языках.
*   **Приглушение музыки** — ForgeTTSEngine автоматически приглушает Apple Music / Spotify во время тренировочной речи и восстанавливает громкость после.
*   **Автоопределение тона** — содержимое сообщения сканируется на эмоциональные ключевые слова (PR, усталость, травма, возвращение и т. д.) и автоматически выбирается подходящий тон.
*   **Голосовые команды** — VoiceCommandEngine обрабатывает запись подходов без рук («записать 100 кг 5 повторений RPE 8») и навигацию («показать мою карту мышц») через фреймворк Speech.

## 🏗️ Архитектура

*   **ForgeCoachCore** — общий Swift-пакет (SPM), содержащий все движки, модели и бизнес-логику. Используется iOS-приложением, Mac Catalyst-приложением и расширением watchOS.
*   **llama.cpp (Metal)** — вывод на устройстве для модели Prism 1.7B GGUF через локальный пакет SPM (`_llama_cpp_local`). Только для iOS/macOS; исключено из watchOS при компиляции.
*   **Мост WatchConnectivity** — `WatchBridge` двунаправленно синхронизирует заряды мышц, заряд тела и флаги функций между iPhone и Watch в реальном времени.
*   **Синхронизация CloudKit** — `CloudKitSyncEngine` реплицирует историю тренировок и профиль пользователя между устройствами с использованием токенов изменений CKRecord и разрешением конфликтов.
*   **HealthKit** — считывает HRV, фазы сна, ЧСС в покое, активную энергию и образцы тренировок. Записывает сводки тренировочных сессий.
*   **Siri Intents** — `LogWorkoutIntent`, `GetBodyBatteryIntent`, `StartRestTimerIntent` для интеграции с Быстрыми командами.

## 🔒 Конфиденциальность

*   **Локальный приоритет** — все биометрические данные остаются на устройстве. Без SDK аналитики. Без сторонних отчётов о сбоях.
*   **HealthKit** — доступ только для чтения, кроме записи тренировочных сессий. Описано в метке конфиденциальности App Store.
*   **ИИ-промпты** — уровень Pro никогда не отправляет данные на какой-либо сервер. Уровень Elite отправляет анонимизированный контекст тренировки на сервер вывода Prism (без PII, без данных HealthKit).
*   **Проверка подписки** — электронная почта отправляется через HTTPS на `api.forgecoach.app/subscription/check`. Электронная почта хранится в Keychain с TTL 24 часа и льготным окном 48 часов.
*   **Данные цикла** — данные двигателя Femme хранятся только в CoreData, никогда не синхронизируются.
